# 🚀 JavaScript Koans - Compass

## 👨‍💻 Sobre o Projeto

Este repositório contém a resolução dos exercícios do **JavaScript Koans**, uma abordagem prática para aprendizado baseada em testes.

O objetivo é evoluir o entendimento de JavaScript através da análise e correção de testes utilizando **Jasmine**, reforçando conceitos fundamentais da linguagem.

Busco evoluir continuamente em JavaScript e automação, fortalecendo a base teórica para aplicar na prática em testes WEB, APP e API.

---

## 🧠 O que foi praticado

Durante a resolução dos Koans, foram abordados conceitos essenciais como:

* Tipos e comparações (`==` vs `===`)
* Arrays e manipulação de dados
* Funções e escopo (lexical scope)
* Objetos e propriedades
* Mutabilidade
* Prototype e herança
* Higher Order Functions (`map`, `filter`, `reduce`)
* Programação funcional
* Manipulação de coleções com Underscore.js

---

## 🔁 Abordagem

Os exercícios seguem o padrão:

* Testes iniciam **falhando (vermelho)**
* O código é ajustado para atender a expectativa
* Objetivo final: todos os testes **passando (verde)**

> Mais do que acertar, o foco foi entender o comportamento do JavaScript em cada cenário.

## 🚀 Como executar

1. Clone o repositório:

```bash
git clone <url-do-repositorio>
```

2. Acesse a pasta:

```bash
cd javascript-koans-<seu-nome>-compass
```

3. Abra o arquivo no navegador:

```bash
KoansRunner.html
```

4. Resolva os testes até ficarem todos verdes ✔
